Vorstellung

Hier können Sie besser verstehen über ganze seite funktionalitäten.

  • An "Alle Filmen" zeigen Sie alle Filmen Danten an.
  • Bei Genre, gehen Sie zu andere Seite mit verschiedene links.
    Es gibt links für daten in HTML und daten in XML format.
  • Die seiten "Regisseure", "Schauspieler", "Genre & Schauspieler" und "Extra Informationen" geben Tabellen auch mit verschiedene informationen.
    Das kommt immer mit zwei daten format. HTML und XML.
    z. B., an der "Extra Informationen" zeigen Sie informationen über frei filmen (für mieten) an. Zeigen Sie auch informationen über kunden, dass eine Mahnung mit Rechnung erhalten haben.
  • Der "Programmausführung" Link richtet zu andere seiten mit viele kode.
    Gibt es kode mit sql skript und verschiedene seite mit shell kode.
    Kode für die Datenbank anlegen, Tabellen anlegen, informationen einfügung und informationen finden.

BESCHREIBUNG

Diese teil ist für kleine informationen in jeden seiten. Wie so kann mann besser verstehen was und wie möglich eine Aktion machen ist.
